Tool for writing 3D text using ASCII characters

As you can see above, 3dwrite helps you to write any string of characters in 3D. It is a python script to which you provide a string and it outputs the ASCII 3D representation of the string.

The 3D representation of each character is pre-defined in a text file. The script reads from this file and then arranges these representations next to each other. Each character's representation was manually designed (using nothing more than a text editor :P ).

Contribute

You can design your own "3D font" if you like. All you need is a good text editor like Sublime which will let you view whitespaces easily. The project is on github so you can contribute to both the script and the font :) .
